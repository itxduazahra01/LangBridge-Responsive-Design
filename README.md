# 🌐 LangBridge - Responsive Web Layout

A responsive version of the LangBridge AI Translator landing page built during my Frontend Development internship.

## 🚀 Project Overview

This project is the second phase of LangBridge where the original static webpage was improved to provide a better user experience across different screen sizes.

The goal was to create a layout that works smoothly on desktop, tablet, and mobile devices.

## ✨ Features

- Fully responsive landing page
- Mobile-friendly navigation layout
- Responsive hero section
- Flexible translator interface
- Responsive feature cards
- Improved spacing and alignment
- Mobile and desktop layouts using CSS media queries

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Media Queries
- Flexbox
- CSS Grid

## 📱 Responsive Design

The website adapts to:
- Desktop screens
- Tablets
- Mobile devices

## 📌 Project Evolution

This is the second version of LangBridge.

Previous:
- LangBridge Landing Page → Static HTML & CSS design

Current:
- LangBridge Responsive → Optimized for multiple screen sizes

Next improvements:
- JavaScript interactions

- Form validation

## 👩‍💻 Learning Outcome

Through this project, I practiced responsive web design concepts and learned how to create layouts that adjust smoothly across different devices.
